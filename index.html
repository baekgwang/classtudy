About Me

안녕하세요 김도영입니다. 저는 전 회사를 다니며 업무 특성상 주로 앱 업데이트 및 수정 사항에 대해 개발 부서와 소통이 많았습니다.
저는 개발 부서에 전달했던 기획 안이 앱이나 웹 상에서 그대로 구현되는 것을 보고 관심이 생겼습니다.
그 잠깐의 관심으로 끝나지 않고 저는 인터넷 참고자료 및 유튜브 영상을 찾아보며 그 분야에 대해 좀더 자세하게 알아봤습니다.
이러한 과정속에 국비 지원으로 개발분야에 공부를 할 수 있다는 것을 알게 되었습니다. 본격적으로 개발 분야를 공부하기 위해 자바 기반 안드로이드앱 개발 과정을 수강하였습니다.
과정에 참여하면서 JAVA 언어 문법을 공부하였고, JAVA 언어를 기반으로 하는 JSP를 이용하여 동적인 웹사이트를 생성하는 법을 배웠습니다.
또한 ORACLE, MY SQL를 사용하여 데이터베이스에 대하여 공부를 하였습니다. 마지막으로 Spring boot를 사용 사용할 때에는 MVC패턴으로 팀 프로젝트를 진행했습니다. 감사합니다.
